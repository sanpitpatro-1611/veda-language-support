# VEDA Language Support

Official Visual Studio Code extension for the **VEDA Programming Language**.

VEDA is a beginner-friendly, human-readable programming language designed to make coding simple, expressive, and easy to learn.

---

## Features

* ✅ Syntax highlighting for VEDA keywords
* ✅ Support for `.veda` files
* ✅ Line comment support using `#`
* ✅ Automatic bracket and quote pairing
* ✅ Language configuration optimized for VEDA
* ✅ Built for VEDA v1.0 ALPHA

---

## Supported Keywords

* `say`
* `remember`
* `ask`
* `calculate`
* `change`
* `if`
* `otherwise`
* `repeat`
* `action`
* `do`

---

## Example

```veda
# Hello World

remember name: "Sanpit"

say: "Welcome to VEDA"

ask age: "Enter your age"

if age greater than 18
    say: "Adult"
otherwise
    say: "Minor"
```

---

## Installation

Install the extension from the Visual Studio Code Marketplace or install the generated `.vsix` package.
